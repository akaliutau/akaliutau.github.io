[GitHub repository](https://github.com/akaliutau/quietplace)

QuietPlace is a safety-aware adaptive semantic noise suppression system. It turns an audio scene into an inspectable controller plan to executed on deterministic ANC hardware, then executes only deterministic, whitelisted audio functions to suppress noise via generating anti-noise. ANC subsystem can be any commercial or open-source software/hardware; for this demo we've built a simple toy ANC to prove the point.

Sumbitted for Google Deepmind Hackathon 2026.
